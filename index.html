OneeChan

A fork of 4chan Style Script, for close maintaining, bug fixing, and quality.

OneeChan is now free as in freedom (and beer!).

Learn how to contribute to OneeChan or submit your own custom theme!

OneeChan has recently switched from userstyle to a userscript. The benefits of this are near endless! Now you no longer have to reinstall to change your options, you can customize OneeChan's colors to your liking right from the page, and there's no cost to performance.

How do I get started?

  1. Install 4chan X -- Mayhem's or ihavenoface's.
  2. Refresh the 4chan board and enable "Persistent QR" and "Autohide QR" in 4chan X's settings.
  3. Follow the instructions below for your browser, and report any issues you find.

Chrome

  1. Install Tampermonkey.
  2. Install OneeChan with Tampermonkey.
  3. If you don't want to use Tampermonkey, install OneeChan the way you installed 4chan X. If you do so, the front page and report page will not respect your theme choice, that's all.

Firefox

  1. Install Greasemonkey (Scriptish is fine, but hasn't been updated since January).
  2. Restart Firefox.
  3. Install OneeChan.

Opera

  • Save as OneeChan.user.js into your userjs folder.
