OneeChan has recently switched from userstyle to a userscript. The benefits of this are near endless! Now you no longer have to reinstall to change your options, you can customize OneeChan's colors to your liking right from the page, and there's no cost to performance.

How do I get started?

Before installing OneeChan, you need to install 4chan X and in its settings enable "Persistent QR." Now, make sure the old OneeChan userstyle is disabled in Stylish if you have it. Follow the instructions below for your browser, and report any issues you find.

Chrome

  1. Add the --enable-easy-off-store-extension-install target flag to your Chrome shortcut.
  2. Restart Chrome.
  3. Install OneeChan.

Firefox

  1. Install Scriptish.
  2. Restart Firefox.
  3. Install OneeChan.

Opera

  1. If you haven't set your userscripts folder:
    1. Go to Preferences (Ctrl + F12) → Advanced → Content → JavaScript Options → User JavaScript Folder.
    2. Chose what will be your userscripts folder.
  2. Save the OneeChan file as OneeChan.js (NOT OneeChan.user.js, to prevent flashing) into your userscripts folder.

Opera Mobile

  1. If you haven't set your userscripts folder:
    1. Enter opera:config in the address bar.
    2. Under User Prefs, enable User JavaScript.
    3. Set User JavaScript File to what will be your userscripts folder, /sdcard/js for example
    4. Save your settings.
  2. Save the OneeChan file as OneeChan.js (NOT OneeChan.user.js, to prevent flashing) into your userscripts folder.
